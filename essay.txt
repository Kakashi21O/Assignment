# 6. Replace all "to" With Your Name


Example:

`to → Mantu`

`git vim`: use ::%s/to/Mantu/g in it to Replace

`git add .`: Add updated file

`git commit -m "Replaced to with Mantu"`: Save changes

`git push`: Upload changes

# 7. Revert Changes


`git log --oneline`: Show commit history

`git revert COMMIT_ID`: Undo old commit

`git push`: Upload reverted changes

# 8. Create New Branch


`git checkout -b new_branch`: Create and switch branch

# 9. Replace "Torvalds" With Your Name
